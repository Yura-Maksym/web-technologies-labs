# 🍫 Confectionery REST API & Web App

Повноцінний вебзастосунок для каталогу кондитерської продукції. Проєкт включає REST API бекенд, клієнтську частину (фронтенд) з авторизацією, покриття тестами та налаштований CI/CD пайплайн.

## 🛠 Технологічний стек
* **Бекенд:** Python 3.11, FastAPI, SQLAlchemy (SQLite)
* **Фронтенд:** HTML5, JS (Fetch API), Bootstrap 5 (UI), `localStorage` (Shared State)
* **Тестування:** Pytest (Coverage > 60%), Selenium (Web Scraping)
* **DevOps / Деплой:** Docker, GitHub Actions (CI), Render (CD)

## 🚀 Функціонал
1. **REST API:** Повноцінний CRUD для сутностей (Продукти, Категорії, Інгредієнти).
2. **Клієнтський інтерфейс:** Три зв'язані сторінки (Логін, Каталог, Профіль).
3. **Авторизація:** Захист маршрутів та збереження стану користувача між переходими.
4. **Автоматизація:** CI/CD конвеєр автоматично тестує та розгортає нові версії з гілки `main`.

## 💻 Локальний запуск
1. `pip install -r requirements.txt` (або вручну: fastapi, uvicorn, sqlalchemy)
2. `uvicorn main:app --reload`
3. Відкрити `http://127.0.0.1:8000`
